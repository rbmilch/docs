SoundGetWaveVolume

Ermittelt die Wave-Wiedergabelautstärke eines Audio-Geräts.

SoundGetWaveVolume, AusgabeVar , Gerätenummer

Parameter

AusgabeVar

Name der Variable, in der die abgerufene Lautstärke gespeichert werden soll, welche eine Gleitkommazahl zwischen 0 und 100 ist. Die Variable wird leer gemacht, wenn ein Problem beim Abrufen der Lautstärke aufgetreten ist. Das Format der Gleitkommazahl, z. B. ihre Dezimalstellen, wird über SetFormat bestimmt.

Gerätenummer

Fehlt dieser Parameter, gilt standardmäßig 1 (das erste Audiogerät), das für gewöhnlich das Standardgerät des Systems für Aufnahme und Wiedergabe ist. Geben Sie eine Zahl größer als 1 an, um ein anderes Audiogerät anzusteuern.

ErrorLevel

[v1.1.04+]: Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Weitere Informationen finden Sie unter Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Die derzeitige Wave-Wiedergabelautstärke kann über SoundSetWaveVolume bestimmt werden. Einstellungen wie Gesamtlautstärke, Synthesizer, Mikrofon, Stummschaltung, Höhe und Bass können mithilfe von SoundSet und SoundGet gesetzt oder abgerufen werden.

[v1.1.10+]: In Windows Vista und höher ist dieser Befehl das gleiche wie SoundGet, wenn Komponententyp auf Wave und Kontrolltyp auf Volume gesetzt wird.

Siehe auch

SoundSetWaveVolume, SoundSet, SoundGet, SoundPlay

Beispiel

SoundGetWaveVolume, AusgabeVar
MsgBox, Die aktuelle Wave-Wiedergabelautstärke ist bei %AusgabeVar%`%.
