EnvDiv

Setzt eine Variable auf sich selbst geteilt durch den angegebenen Wert.

Veraltet: Dieser Befehl ist nicht für neue Skripte empfohlen. Verwenden Sie stattdessen Var := Var / Wert oder Var /= Wert.

EnvDiv, Var, Wert

Parameter

Var

Name der Variable, bei der die Operation angewendet werden soll.

Wert

Ein Integer, eine Gleitkommazahl oder ein Ausdruck.

Bemerkungen

Dieser Befehl entspricht der Kurzschreibweise: Var /= Wert.

Eine Division durch Null führt beim Laden des Skripts zu einer Fehlermeldung (falls möglich); ansonsten wird die Variable leer gemacht.

Wenn Var oder Wert leer ist oder nicht mit einer Zahl beginnt, wird der Wert bei der Berechnung als 0 behandelt (außer bei interner Verwendung in einem Ausdruck wie Var := X /= Y).

Enthält Var oder Wert einen Dezimalpunkt, ist das Endergebnis eine Gleitkommazahl, abhängig vom via SetFormat gesetzte Format. Ansonsten wird das Ergebnis gekürzt (z. B. würde 19 geteilt durch 10 gleich 1 ergeben).

Siehe auch

EnvAdd, EnvSub, EnvMult, SetFormat, Ausdrücke, If Var is [not] Typ, SetEnv, bitweise Operationen (Transform)

Beispiel

EnvDiv, MeineZählung, 2
MeineZählung /= 2  ; das gleiche wie oben
