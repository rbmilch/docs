GetKeyName / GetKeyVK / GetKeySC [v1.1.01+]

Ermittelt den Namen/Text, virtuellen Tastencode oder Scancode einer Taste.

Zeichenkette := GetKeyName(Taste)
Nummer := GetKeyVK(Taste)
Nummer := GetKeySC(Taste)

Parameter

Taste

Ein VK oder SC-Code, wie z. B. "vkA2" oder "sc01D", eine Kombination von beiden oder ein Tastenname. Zum Beispiel wird sowohl GetKeyName("vk1B") als auch GetKeyName("Esc") Escape zurückgeben, während GetKeyVK("Esc") 27 zurückgibt. Beachten Sie, dass VK- und SC-Codes hexadezimal sein müssen. Um eine dezimale Nummer in das entsprechende Format umzuwandeln, verwendet man Format("vk{:x}", vk_code) oder Format("sc{:x}", sc_code).

Rückgabewert

Diese Funktionen geben den Namen, virtuellen Tastencode oder Scancode von Taste zurück.

Siehe auch

GetKeyState, Tastenliste, Format

Beispiele

; Zeigt Informationen über eine bestimmte Taste an.
Taste := "LWin" ; Kann eine beliebige Taste sein.

name  := GetKeyName(Taste)
vk    := GetKeyVK(Taste)
sc    := GetKeySC(Taste)

MsgBox, % Format("Name:`t{}`nVK:`t{:X}`nSC:`t{:X}", name, vk, sc)
