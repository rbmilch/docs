ObjAddRef() / ObjRelease() [AHK_L 53+]

Erhöht oder verringert die Referenzzählung eines Objekts.

ObjAddRef(Ptr)
ObjRelease(Ptr)

Parameter

Ptr

Ein nicht verwalteter Objekt-Pointer oder COM-Interface-Pointer.

Rückgabewert

Diese Funktionen geben die neue Referenzzählung zurück. Dieser Wert sollte nur zum Debuggen verwendet werden.

Siehe auch

Referenzzählung

Zwar beziehen sich die folgenden Artikel auf die Referenzzählung von COM, allerdings decken sie einige wichtige Konzepte und Regeln ab, die im Allgemeinen auch für AutoHotkey-Objekte gelten: IUnknown::AddRef, IUnknown::Release, Regeln bei der Referenzzählung.

Beispiele

Siehe ComObjConnect.

obj := Object()

; Die folgenden zwei Zeilen sind gleichwertig:
ptr1 := Object(obj)
ptr2 := ObjectToPointer(obj)

ObjectToPointer(obj) {
    if !IsObject(obj)
        return ""
    ptr := &obj

    ObjAddRef(ptr)
    return ptr
}

; Jeder ermittelter Pointer via Object() oder ObjectToPointer() muss manuell freigegeben werden,
; so dass das Objekt frei gemacht und der verbrauchte Speicher wiederverwendet werden kann.
ObjRelease(ptr2)
ObjRelease(ptr1)
