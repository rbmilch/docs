Asc()

Gibt den numerischen Wert des ersten Bytes oder der ersten UTF-16-Codeeinheit in der angegebenen Zeichenkette zurück.

Nummer := Asc(Zeichenkette)

Parameter

Zeichenkette

Eine Zeichenkette, deren numerischer Wert abgerufen werden soll.

Rückgabewert

Diese Funktion gibt einen numerischen Wert von 0 bis 255 (bei ANSI) oder von 0 bis 0xFFFF (bei Unicode) zurück. Einzelheiten finden Sie unter Unicode vs ANSI. Wenn Zeichenkette leer ist, wird sie eine 0 zurückgeben.

Bemerkungen

Diese Funktion ist äquivalent zu Transform, AusgabeVar, Asc.

Für Unicode-Ergänzungszeichen können Sie stattdessen Ord(Zeichenkette) verwenden.

Siehe auch

Transform, Ord(), Chr()

Beispiele

; Beide unteren Message Boxes werden 116 anzeigen, weil nur das erste Zeichen berücksichtigt wird.
MsgBox, % Asc("t") 
MsgBox, % Asc("test")
