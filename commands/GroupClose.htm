GroupClose

Schließt das aktive Fenster, das gerade mit GroupActivate oder GroupDeactivate aktiviert worden ist, und aktiviert das nächste Fenster in der Gruppe. Kann auch alle Fenster in einer Gruppe schließen.

GroupClose, Gruppenname , A|R

Parameter

Gruppenname

Name der Gruppe, die ursprünglich mit GroupAdd definiert worden ist.

A|R

Ist der Buchstabe A angegeben, werden alle Fenster in der Gruppe geschlossen. Diese Methode hat den gleichen Effekt wie WinClose ahk_group Gruppenname.

Ansonsten: Schließt der Befehl das aktive Fenster, wird das nächste Fenster in der Reihe aktiviert. Dieser Parameter legt fest, ob das älteste oder das neueste Fenster aktiviert werden soll, wenn kein einziges Fenster aus der Gruppe aktiv ist. Fehlt dieser Parameter, wird immer das älteste Fenster aktiviert. Ist der Buchstabe R angegeben, wird das neueste (zuletzt aktive) Fenster aktiviert, aber nur, wenn beim Ausführen des Befehls kein Fenster aus der Gruppe aktiv ist. "R" ist beispielsweise nützlich, wenn Sie vorübergehend eine andere Aufgabe durchführen wollen. Wenn Sie dann mit GroupActivate, GroupDeactivate oder GroupClose zur Gruppe zurückkehren, wird nicht das älteste, sondern das aktuell genutzte Fenster aktiviert.

Bemerkungen

Enthält der A|R-Parameter kein "A", wird das Verhalten des Befehls davon abhängig, ob die vorherige Aktion auf Gruppenname GroupActivate oder GroupDeactivate war. Wenn es GroupDeactivate war, wird das aktive Fenster nur geschlossen, wenn das Fenster nicht zur Gruppe gehört (ansonsten passiert nichts). Wenn es GroupActivate oder keiner dieser Befehle war, wird das aktive Fenster nur geschlossen, wenn das Fenster zur Gruppe gehört (ansonsten passiert nichts). Durch dieses Verhalten kann GroupClose mit einem Hotkey verbunden werden, als Ergänzung zum GroupActivate- oder GroupDeactivate-Hotkey von Gruppenname.

Siehe GroupAdd für mehr Details über Fenstergruppen.
