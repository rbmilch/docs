IsFunc() [v1.0.48+]

Gibt eine Nummer ungleich 0 zurück, wenn die angegebene Funktion im Skript vorkommt.

MinParamsPlus1 := IsFunc(Funktionsname)

Parameter

Funktionsname

Name der Funktion, deren minimale Anzahl von Parametern abgerufen werden soll. Funktionsname muss explizit im Skript vorkommen. In [v1.1.00+] kann Funktionsname ein Funktionsverweis anstelle eines Namens sein.

Rückgabewert

Diese Funktion gibt die minimale Anzahl an Parametern zurück, addiert mit 1 (also 1 für eine Funktion ohne Parameter, 2 für eine Funktion mit einem Parameter und so weiter). Wenn Funktionsname im Skript nicht explizit vorhanden ist (z. B. durch #Include oder einen nicht-dynamischen Aufruf einer Library-Funktion), wird sie eine 0 zurückgeben.

Siehe auch

Dynamischer Aufruf einer Funktion, Funktionsverweise, Func-Objekt, Func(), A_ThisFunc

Beispiele

Anzahl := IsFunc("RegExReplace") ; Hier kann eine beliebige Funktion benutzt werden.
if Anzahl
  MsgBox, % "Diese Funktion existiert und hat " Anzahl-1 " Pflichtparameter."
else
  MsgBox, % "Diese Funktion existiert nicht."
