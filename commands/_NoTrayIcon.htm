#NoTrayIcon

Deaktiviert die Darstellung des Tray-Symbols.

#NoTrayIcon

Diese Direktive irgendwo im Skript verhindert, dass das Tray-Symbol beim Starten des Skripts angezeigt wird (auch dann, wenn das Skript kompiliert ist).

Wenn Sie diese Direktive bei einem Skript nutzen, das Hotkeys hat, empfiehlt es sich, einen Hotkey an den ExitApp-Befehl zu binden. Ansonsten würde der Fall eintreten, dass es keine einfache Möglichkeit gibt, das Programm zu beenden (ohne dabei den Computer neu zu starten oder den Prozess zu beenden). Zum Beispiel: #x::ExitApp.

Mit Menu, Tray, Icon oder Menu, Tray, NoIcon kann das Tray-Symbol jederzeit ein- oder ausgeblendet werden, während das Skript ausgeführt wird. Wenn Sie Menu, Tray, NoIcon im oberen Bereich des Skripts nutzen, werden Sie feststellen, dass das Tray-Symbol beim Starten des Skripts kurz sichtbar ist. Um das zu verhindern, können Sie stattdessen #NoTrayIcon verwenden.

Die interne Variable A_IconHidden enthält 1, wenn das Tray-Symbol momentan ausgeblendet ist, andernfalls 0.
