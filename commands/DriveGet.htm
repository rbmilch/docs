DriveGet

Ruft verschiedene Arten von Informationen über die Laufwerke des Computers ab.

DriveGet, AusgabeVar, Befehl , Wert

Parameter

AusgabeVar

Name der Variable, in der das Ergebnis von Befehl gespeichert werden soll.

Befehl, Wert
Siehe untere Liste.

Befehl, Wert

Die Parameter Befehl und Wert sind voneinander abhängig und können wie folgt genutzt werden. Bei Misserfolg wird AusgabeVar leer gemacht und ErrorLevel auf 1 gesetzt.

List [, Typ]: Speichert in AusgabeVar eine Reihe von Buchstaben, die jeweils einen Laufwerksbuchstaben im System repräsentieren. Zum Beispiel: ACDEZ. Wenn Typ fehlt, werden alle Laufwerkstypen abgerufen. Geben Sie ansonsten für Typ eines der folgenden Wörter an, um nur einen bestimmten Laufwerkstyp abzurufen: CDROM, REMOVABLE, FIXED, NETWORK, RAMDISK, UNKNOWN.

Capacity (oder Cap), Pfad: Ermittelt die Gesamtkapazität von Pfad (z. B. C:\) in Megabyte. Mit DriveSpaceFree kann der freie Speicherplatz abgerufen werden.

Filesystem (oder FS), Laufwerk: Ruft den Typ des Dateisystems von Laufwerk ab. Laufwerk ist der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und einem umgekehrten Schrägstrich, oder ein Netzwerkpfad im UNC-Format wie \\server1\share1. AusgabeVar wird auf eines der folgenden Wörter gesetzt: FAT, FAT32, NTFS, CDFS (typisch für CDs), UDF (typisch für DVDs). AusgabeVar wird leer gemacht und ErrorLevel auf 1 gesetzt, wenn das Laufwerk kein formatiertes Medium enthält.

Label, Laufwerk: Ruft die Datenträgerbezeichnung von Laufwerk ab. Laufwerk ist der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und einem umgekehrten Schrägstrich, oder ein Netzwerkpfad im UNC-Format wie \\server1\share1. Die Bezeichnung kann wie folgt geändert werden: Drive, Label, C:, MeineBezeichnung.

Serial, Laufwerk: Ruft die Datenträgerseriennummer von Laufwerk als dezimalen Integer ab. Laufwerk ist der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und einem umgekehrten Schrägstrich, oder ein Netzwerkpfad im UNC-Format wie \\server1\share1. Siehe SetFormat, wie man diesen Wert in einen hexadezimalen Wert umwandeln kann.

Type, Pfad: Ruft den Laufwerkstypen von Pfad ab, der eines der folgenden Wörter ist: Unknown, Removable, Fixed, Network, CDROM, RAMDisk.

Status, Pfad: Ruft den Status von Pfad ab, der eines der folgenden Wörter ist: Unknown (könnte auf unformatiert/RAW hinweisen), Ready, NotReady (typisch für leere Wechseldatenträger), Invalid (Pfad existiert nicht oder ist ein Netzwerklaufwerk, auf das momentan nicht zugegriffen werden kann, usw.)

StatusCD [, Laufwerk]: Ruft den Medienstatus eines CD- oder DVD-Laufwerks ab. Laufwerk ist der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt (wenn Laufwerk fehlt, wird das Standard-CD/DVD-Laufwerk verwendet). AusgabeVar wird leer gemacht, wenn der Status nicht abgerufen werden kann. Ansonsten wird es auf eine der folgenden Zeichenketten gesetzt:

not ready Das Laufwerk ist nicht bereit für den Zugriff, zum Beispiel aufgrund eines Schreibvorgangs. Bekannte Einschränkung: "not ready" tritt auch auf, wenn das Laufwerk eine DVD statt CD enthält.
open Das Laufwerk enthält keine CD, oder ist ausgefahren.
playing Das Laufwerk spielt gerade eine CD ab.
paused Die zuvor abgespielte Audio- oder Videowiedergabe ist nun pausiert.
seeking Das Laufwerk ist am Suchen.
stopped Das Laufwerk enthält eine CD, aber greift momentan nicht darauf zu.

Dieser Befehl wird eventuell nicht funktionieren, wenn das angegebene Laufwerk ein Netzwerk- oder Nicht-CD/DVD-Laufwerk ist; wenn dieser Befehl in solchen Fällen oder aus einem anderen Grund fehlschlägt, wird AusgabeVar leer gemacht und ErrorLevel auf 1 gesetzt.

Wenn das Laufwerk vor Kurzem geschlossen wurde, kann es zu einer Verzögerung kommen, bevor der Befehl zu Ende ist.

Mit dem Drive-Befehl kann ein Laufwerk ein- oder ausgefahren werden.

ErrorLevel

[v1.1.04+]: Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Weitere Informationen finden Sie unter Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.
