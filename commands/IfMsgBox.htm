IfMsgBox

Überprüft, welchen MsgBox-Button der Benutzer gedrückt hat.

IfMsgBox, ButtonName

Parameter

ButtonName

Eine der folgenden Zeichenketten, die anzeigen soll, welchen MsgBox-Button der Benutzer gedrückt hat:

Yes
No
OK
Cancel
Abort
Ignore
Retry
Continue [v1.0.44.08+]
TryAgain [v1.0.44.08+]
Timeout (wenn MsgBox die Zeit überschreitet)

Siehe auch

MsgBox

Beispiel

MsgBox, 4, , Möchten Sie fortfahren?, 5  ; 5-sekündliche Zeitüberschreitung.
IfMsgBox, No
    Return  ; Benutzer hat den Button "Nein" gedrückt.
IfMsgBox, Timeout
    Return ; d. h. "Nein", wenn Zeitüberschreitung.
; Ansonsten fortsetzen:
